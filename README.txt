- En el menu principal, se adiciono la funcionalidad para ver todos los jugadores de la base de datos, acumulando sus puntos y cantidad de movimientos historicos.

Notar que al ejecutar por primera vez el servidor, apareceran dos mensajes de error sobre la apertura de archivos, tanto ranking como jugadores. 
Esto se debe a que no estan creados, por lo que luego, ya no se repite el mensaje.
